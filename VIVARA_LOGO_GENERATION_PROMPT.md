# Vivara Logo Generation Prompts

## Main Logo (Horizontal Wordmark)

```
Create a modern, elegant app logo for "Vivara" - a video editing app
that makes people radiate in vivid color while the background becomes grayscale.

Design requirements:
- The word "Vivara" with "Viv" in a warm electric coral to vivid purple
  gradient (#FF6B6B to #845EF7)
- "ara" in sophisticated charcoal gray (#374151)
- Clean, modern sans-serif typeface (similar to SF Pro Display or Montserrat)
- Slightly rounded letterforms for warmth and approachability
- Optional: subtle luminous glow or aura effect around the "V" or entire word
- Overall feel: elegant, warm, premium, radiant
- Vector-style, clean lines, suitable for light and dark backgrounds
- Aspect ratio: horizontal logo, roughly 3:1 width to height
```

## App Icon — Version 1 (Radiant Silhouette)

```
Create an iOS app icon for "Vivara" - a video editing app:

Design concept:

- Soft grayscale background (#F3F4F6)
- A person silhouette in the center, filled with radiant gradient colors
  (electric coral #FF6B6B → vivid purple #845EF7 → luminous cyan #22D3EE)
- Subtle luminous "aura" glow effect emanating from the figure
- The figure should appear to radiate warmth and light
- Clean, elegant, minimal design
- Must look sharp at small sizes (like iPhone home screen)
- Should feel warm and inviting, not cold or technical
- 1024x1024 resolution for app store
```

## Quick Prompt for Fast Generation (Midjourney/DALL-E style)

**For full logo:**
```
Modern elegant app logo design, wordmark "Vivara", "Viv" in coral to purple
gradient, "ara" in gray, subtle luminous glow effect, warm premium aesthetic,
vector style, clean typography, app store ready --ar 3:1 --v 6
```

**For app icon:**
```
iOS app icon, person silhouette radiating warm coral purple cyan gradient,
soft gray background, subtle aura glow effect, elegant minimal design,
rounded square, premium app aesthetic, 1024x1024 --v 6
```

## Recommended AI Tools & Settings

### Midjourney
- Use `--v 6` or `--v 6.1` for best quality
- Add `--style raw` for more literal interpretation
- Use `--ar 1:1` for app icon, `--ar 3:1` for horizontal logo

### DALL-E 3
- Use detailed prompts above
- Specify "vector illustration style" or "flat design style"
- Request "professional app branding" in the prompt

### Adobe Firefly
- Use "graphic design" style
- Enable "vector" or "illustration" mode
- Good for variations once you have a base concept

### Stable Diffusion (with ControlNet)
- Use models trained on logo design (like "logo-diffusion")
- ControlNet can help with precise composition
- Good for iteration and fine-tuning

## Tips for Best Results

1. **Generate multiple variations** - try different styles, compositions, color balances
2. **Test at small sizes** - app icons must be legible at 60x60px
3. **Try both literal and abstract** - some generators work better with conceptual descriptions
4. **Iterate on promising results** - use img2img or variations once you find a direction you like
5. **Consider hiring a designer** for final polish if AI gets close but not perfect
